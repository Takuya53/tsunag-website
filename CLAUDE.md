# プロジェクトルール（TSUNAG コーポレートサイト）

## ブランド

- **会社名**: 株式会社TSUNAG
- **Mission**: 「豊かさを、つなぐ。」
- **Vision**: AIとクリエイティブの力で、中小企業の未来を豊かにする。

### ブランドキーワード

- 高級感
- 未来感
- 信頼感
- シンプル
- 余白
- Apple
- Aesop
- Minimal

上記キーワード（特にApple / Aesopのようなミニマル・高級ブランドのトーン）を、コピーライティング・レイアウト・余白設計・アニメーションの判断基準とすること。

### デザインカラー

- White
- Black
- Gold

このベースカラー（White / Black / Gold）を軸に配色すること。新しい色を追加する場合はブランドキーワードとの整合性を確認する。

### 禁止事項

- 安売り感が出る表現・演出
- 派手すぎる演出（過剰なアニメーション、けばけばしい配色）
- チープなUI（安っぽいボタン、素材感のない装飾）
- 古いWebデザイン（角丸のない箱組み、古臭いグラデーション、時代遅れのレイアウトパターン）

## 技術

- Next.js
- TypeScript
- Tailwind CSS

## 開発ルール

- 実装前に必ず計画を提示する
- 既存デザインを壊さず改善する
- レスポンシブ対応を必須とする
- SEOを維持する（メタデータ、構造化データ、sitemap/robots等の既存対応を損なわない）
- アクセシビリティを維持する
- コンポーネントを再利用する（新規のベタ書きを避け、`app/components/sections` 配下の既存パターンに揃える）
- Git Commitは指示があるまで行わない
